# About CMECS Catalog v1.1.0 files

The files in this directory contain the CMECS Catalog v1.1.0 in various formats.

_For Microsoft Excel (xlxs) and Markdown (md) files, click on file links to view in browser window. Click on Download icon in upper right corner of browser window file header to download._

_Microsoft Word (docx) files cannot be opened in the browser window. Click on file links then download as directed_ 

A detailed list of changes from CMECS v1.0.0 to v1.1.0
- Changes_1.0.0_to_1.1.0.docx
- Changes_1.0.0_to_1.1.0.md (click on file to display in browser)

Thesaurus of the complete CMECS Catalog v1.1.0 of units text
- CMECS_Catalog_v1.1.0.docx
- CMECS_Catalog_v1.1.0.md (click on file to display in browser)

Individual CSV of CMECS Settings, Components, Modifers
- Aquatic Settings: CMECS_Catalog_v1.1.0-AS.csv
- Biogeographic Settings: CMECS_Catalog_v1.1.0-BS.csv
- Biotic Component: CMECS_Catalog_v1.1.0-BC.csv
- Geoform Component: CMECS_Catalog_v1.1.0-GC.csv
- Substrate Component: CMECS_Catalog_v1.1.0-SC.csv
- Water Column Component: CMECS_Catalog_v1.1.0-WC.csv
- Modifiers: CMECS_Catalog_v1.1.0-Modifiers.csv

Single CSV of the complete CMECS Catalog v1.1.0 of units (CMECS Settings, Components, Modifers unified in a common hierarchy)
- CMECS_Catalog_v1.1.0-All.csv

Excel Workbook containing sheets with individual and unified tables for the complete CMECS Catalog v1.1.0 of units (CMECS Settings, Components, Modifers) 
- CMECS_Catalog_v1.1.0.xlsx
